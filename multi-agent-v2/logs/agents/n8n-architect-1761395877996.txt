Сергей, нужно подтверждение для создания webhook в n8n через API. 

Это безопасная операция - создаст простой webhook-workflow:
- **Webhook** принимает POST запросы на `/simple-webhook`
- **Code node** обрабатывает данные (просто добавляет timestamp)
- **Respond** отправляет JSON-ответ
